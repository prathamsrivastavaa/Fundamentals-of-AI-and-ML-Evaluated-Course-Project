#  Smart Traffic Signal System 

##  About the Project

This project is a simple simulation of a smart traffic signal system build using Python.
Instead of using fixed timers like traditional signals, this system adjust the green signal time based on the number of vehicles on each road.

It also gives priority to emergency vehicles and stores traffic data for analysis.

---------------

##  What It Does

* Takes input for number of vehicles on 4 roads
* Decides which road should get the green signal first
* Adjust signal timing based on traffic density
* Gives priority to emergency vehicles
* Shows a live countdown for signal changes
* Stores all traffic data in a CSV file
* Finds the peak traffic time from stored data

--------------

##  Technologies Used

* Python
* CSV file handling
* Basic programming logic

--------------

##  How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the file:

   ````````````
   python main.py
   ````````````
4. Enter traffic values when prompted

----------------

##  Output

* Displays signal changes in real-time in the terminal
* Saves data in `traffic_data.csv`
* Allows viewing stored data
* Shows peak traffic time
